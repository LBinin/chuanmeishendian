新闻：“记与思”
执笔记思湘大
新闻类报名表
  1. 记者栏目
    1. 报名方式：线上提交参赛材料和信息至指定邮箱，切记上交作品时做好备注并打包。材料类型为新闻消息、人物通讯、新闻调查及新闻评论四大类中四选二，每篇稿件字数不少于500字（参赛作品独立创作，不得抄袭，一旦发现，即取消参赛资格）
    2. 比赛形式：
      1. 学生评委从众多参赛选手上交材料中选出优秀选手20位；
      2. 20位选手进入复赛，现场选出前6位；
      3. 最后6位选手在最后颁奖晚会上现场根据几段视频进行报道，由评委老师评分（占80%）以及现场观众投票（占20%）决定；
      4. 湘大电视台、湘大校报、湘大之声、三翼工作室、青年传媒中心、文心报可推荐1到2名选手，但是同样需要提交材料。

        5. （推荐名额需要在初赛作品提交文件上备注，例如：文心报推荐选手xxx。）

    3. 评审团：
      1. 评选标准：评审团评级
      2. 评审老师：文新院新闻专业老师、校报老师
      3. 学生评委：校报、电视台、湘大之声、三翼工作室、青年传媒中心、文心报各出2人组成评审团
      4. 选出人数：12人
    4. 栏目时间轴：
      4月15日开始报名
      4月28日中午12:00截止报名
      5月3日学生评委初次筛选
      5月5日北山复赛比试
      5月6日公布获奖决赛名单并通知参赛人员
      5月13号晚会决赛并颁奖
  2. 新媒体运营
    1. 作品形式：
      1. 组织、院系新媒体类（各大院系及校级组织强制参赛）
      2. 个人新媒体类（参赛形式为营销策划和运营方案两类）
    2. 作品主题限定：“记与思”（执笔记思湘大）
    3. 报名方式：在指定邮箱提交一份电子档的资料；切记上交作品时做好备注并打包。（参赛者可为各校级组织、各院系、各社团运营公众号，也可以是团队运营较好的公众号，鼓励用个人公众号来参赛，切记参赛作品需独立创作，不得抄袭，一旦发现，即取消参赛资格。）
    4. 评比标准：
      1. 大众评委（占50%）：
        1. 组织、院级：关注数据+创意文章+内容可用性+情怀；
        2. 个人：①营销策划类②运营方案类
      2. 线上第一次评比（占15%：设置“喜欢”和“不喜欢”键）＋晚会现场线上第二次投票（占35%：晚会进行同时进行线上投票）
    5. 栏目时间轴：
      4月15日开始报名
      4月28日中午12:00截止报名
      5月3日开始进行大众评委评比以及线上评比
      5月13日晚会现场进行网站现场投票评比及结果公布并颁奖

视频：“立与行”
湘大人的担当与风采
视频类报名表
  1. 作品形式：
    1. 围绕主题“立与行”，拍摄展示湘大人的担当与风采的小视频（不限于湘大学生，也可以是教职工、校友等）
    2. 作品提交格式为：MOV、MP4等；作品分辨率不低于 1280*720 ，画面速度不低于每秒25帧。
    3. 参赛作品长度不得超过5分钟。
  2. 报名方式：线上提交电子稿以及基本信息至指定邮箱，切记上交作品时做好备注并打包。
  3. 比赛形式：
    1. 把视频放在网站上一周时间，评比根据专业角度、点击量、点赞数、弹幕数等评比（其中专业角度占70%，评分标准总分为100分，评分标准和要求如下：
      ①作品内容（40分）要求：立意鲜明、健康积极、富有感染力、传播力度高、社会影响力好。
      ②拍摄（25分）要求：画面清晰、曝光准确、色彩无失真、构图美观、镜头稳定。
      ③编辑（10分）要求：结构流畅，配乐得当，字幕规范，无跳轴夹帧等瑕疵。
      ④创新性（25分）要求：剧本、拍摄、编辑等方面具有新颖的角度和手法。
    2. 在颁奖晚会播放优秀视频（根据时间长短定）
  4. 栏目时间轴：
    4月15日开始报名
    4月28日中午12:00截止报名
    4月29日-5月2日参赛选手视频作品上传
    5月3-5日参赛选手视频作品评比
    5月6日获奖作品公示并通知获奖者
    5月13号颁奖晚会展出优秀获奖作品
